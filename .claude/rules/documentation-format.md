# 规则名称
文档格式规范

# 规则摘要
书写文档优先使用 Markdown 格式。

# 规则具体内容

## 格式优先原则
- **优先使用 Markdown（`.md`）格式**
- 避免使用 Word、PDF 等不利于版本控制的格式
- 技术文档、设计文档、接口说明、操作手册等均适用

## 例外情况
- 需要特定排版或打印时，可导出 PDF 作为附件
- 复杂图表可配合 Mermaid、PlantUML 嵌入 Markdown
- 外部对接要求其他格式时，以对接要求为准

## 文件组织
- 文档统一存放在 `docs/` 或项目约定的文档目录下
- 临时分析文档按《文件与目录规范》放入 `tmp/docs/`

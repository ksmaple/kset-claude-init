# 规则名称
规则自主优化

# 规则摘要
反复出现的未被文档化的约束，自动提炼为本地规则。

# 规则具体内容

## 触发条件
同时满足以下两点时触发：
1. 该约束尚未写入 `.claude/rules/` 或 `.claude.local/rules/`
2. 在本次对话中被引用/执行 **3 次及以上**，或在多个独立任务中持续出现

## 自动生成流程
1. 识别并总结反复出现的约束
2. 按三段式结构撰写（规则名称、规则摘要、规则具体内容）
3. 写入 `.claude.local/rules/<kebab-case名称>.md`
4. 更新 `CLAUDE.local.md` 的本地配置索引
5. 向用户简要说明生成的规则

## 存放位置
- 一律自动放入 `.claude.local/rules/`（个人本地配置）
- 用户明确要求提升为团队通用约定时，再迁移至 `.claude/rules/`
